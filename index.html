Утилита для конверции excle таблицы в базу данных mysql

Excel в mysql.

ExcelToMySql - утилита написанная на Python (необходимые решения для реализации см. в описании), представляет из себя GUI графический интерфейс под Windows. Удобно, если вам необходимо перенести данные таблицы MS Excel (.xlsx), прайс-лист или перечень чего-либо с описанием, большого объема в базу данных MySQL.
